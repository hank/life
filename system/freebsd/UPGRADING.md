# Normal updating
Alias `locked` to `pkg info -ak|grep yes`

    * `freebsd-update fetch install`
    * `pkg upgrade`
    * `portsnap fetch update`
    * `locked`
    * `pkg unlock <pkgs>`
    * `portmaster -adG`
    * `pkg lock <pkgs>`

# Upgrading releases

    * `freebsd-update upgrade -r 10.3-RELEASE`
